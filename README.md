# ❌⭕ Jogo da Velha (Tic-Tac-Toe)

Uma implementação interativa e moderna do clássico **Jogo da Velha**, desenvolvida com **HTML5, CSS3 e JavaScript puro (Vanilla JS)**.

🔗 **Deploy da aplicação**: [https://jogodavelha-delta.vercel.app/](https://jogodavelha-delta.vercel.app/)

---

## 🎯 Sobre o Projeto

O projeto foi criado para exercitar conceitos fundamentais de desenvolvimento web front-end, como **manipulação do DOM**, **gerenciamento de estados em JavaScript** e **design responsivo**.

O jogo permite a disputa entre dois jogadores no mesmo dispositivo, alternando as jogadas entre "X" e "O", identificando automaticamente condições de vitória ou empate (velha) e permitindo reiniciar a partida instantaneamente.

### 🛠️ Funcionalidades

- **Alternância Automática de Turnos**: Controle dinâmico entre os jogadores 'X' e 'O'.
- **Verificação de Vitória e Empate**: Validação de todas as combinações possíveis a cada jogada.
- **Placar/Status da Partida**: Mensagens visuais indicando a vez do jogador, vitória ou empate.
- **Reinício Rápido**: Botão para resetar o tabuleiro sem recarregar a página.
- **Layout Responsivo**: Interface fluida para dispositivos móveis, tablets e computadores.

---

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do tabuleiro e elementos de interface.
- **CSS3**: Layout com Grid/Flexbox, estilos visuais e responsividade.
- **JavaScript (ES6+)**: Lógica do jogo, checagem de matrizes de vitória e manipulação do DOM.
- **Vercel**: Plataforma de hospedagem e deploy contínuo.
